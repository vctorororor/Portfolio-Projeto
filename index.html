TaskFlow: Gerenciador de Tarefas

Este projeto é uma Single Page Application (SPA) desenvolvida com HTML5, CSS3 e JavaScript puro. O objetivo foi criar uma interface limpa e intuitiva para o gerenciamento de tarefas do dia a dia.

A principal funcionalidade implementada é a persistência de dados. Utilizei a API de localStorage do navegador para garantir que as tarefas do usuário não se percam ao fechar ou recarregar a página. Isso proporciona uma experiência de usuário contínua e eficaz.

A manipulação dinâmica do DOM com JavaScript foi fundamental para criar a interatividade. Funções para adicionar, excluir e marcar tarefas como concluídas atualizam a interface em tempo real, sem a necessidade de recarregar a página. O CSS foi usado para criar um design responsivo, que se adapta bem a desktops e dispositivos móveis.

Análise de Vendas com Python

Neste projeto, mergulhei no mundo da ciência de dados utilizando Python e suas poderosas bibliotecas. O objetivo era analisar um conjunto de dados de vendas para identificar padrões, produtos mais vendidos e o desempenho de vendas ao longo do tempo.

Utilizei a biblioteca Pandas para carregar, limpar e manipular os dados. A etapa de limpeza foi crucial para tratar valores ausentes e garantir a consistência das informações antes da análise.

Com os dados prontos, usei Matplotlib e Seaborn para criar visualizações claras e informativas, como gráficos de barras, gráficos de linha e heatmaps. Essas visualizações foram essenciais para comunicar os insights encontrados de forma eficaz. O projeto foi desenvolvido em um ambiente de Jupyter Notebook para facilitar a exploração iterativa dos dados.

Bot Moderador para Discord

Este projeto foi minha introdução à automação e ao desenvolvimento de aplicações backend com Python. O objetivo era criar um bot funcional para um servidor do Discord, capaz de automatizar tarefas de moderação e aumentar o engajamento da comunidade.

Utilizei a biblioteca discord.py, um wrapper de API assíncrono que facilita a comunicação com a API do Discord. Implementei funcionalidades como:

  • Mensagens de boas-vindas: Saudar novos membros em um canal específico.
  • Sistema de cargos: Permitir que usuários se autoatribuam cargos através de reações.
  • Comandos personalizados: Comandos simples, como exibir regras do servidor ou links úteis.

A importância deste projeto foi aprender a trabalhar com APIs externas, lidar com programação assíncrona (async/await) e entender como gerenciar um projeto que roda continuamente em um servidor.
